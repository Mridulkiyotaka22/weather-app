# 🌤️ Weather App

A clean and minimal weather app built using HTML, CSS, and JavaScript.
It allows you to quickly check real-time weather with a smooth, mobile-style UI.

---

## 🚀 Live Demo

👉 https://weather-app-sandy-pi-75.vercel.app/

*(Deployed on Vercel)*

---

## ✨ Features

* 🔍 Search any city with live suggestions
* 📍 Detect your current location automatically
* 🌡️ Real-time temperature and weather conditions
* 🌤️ Smart icons (day/night + weather-based)
* 🌈 Dynamic background that changes with weather
* 📱 Clean and modern mobile-like UI
* 🌍 Quick access to top cities

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeather API
* Vercel (Deployment)

## ⚙️ Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Mridulkiyotaka22/weather-app.git
```

2. Navigate into the folder:

```bash
cd weather-app
```

3. Open in browser:

* Open `index.html` directly
  OR
* Use Live Server in VS Code

---

## 🌐 Deployment

This project is deployed using Vercel:

👉 https://weather-app-sandy-pi-75.vercel.app/

To deploy your own version:

1. Push your code to GitHub
2. Go to https://vercel.com
3. Import your repository
4. Click **Deploy**

---

## 🔑 API Configuration

This app uses OpenWeather API.

To use your own API key, update:

```js
const API_KEY = "your_api_key_here";
```
